# 📅 Interactive Wall Calendar

🔗 **Live Demo:** https://calendar-project-a3kr.vercel.app

A modern and interactive wall calendar web application built using **Next.js**, **React**, and **Tailwind CSS**.
It allows users to select date ranges and attach notes with persistent storage.

---

## 🚀 Features

* 📅 Dynamic calendar with month navigation
* 🎯 Date range selection (start → end)
* 🔄 Reverse selection handling
* 📝 Notes system with localStorage (data persists after refresh)
* 🎨 Beautiful UI with dark theme
* ⚡ Smooth animations using Framer Motion
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

* **Next.js**
* **React**
* **Tailwind CSS**
* **Framer Motion**
* **date-fns**

---
